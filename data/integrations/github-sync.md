# GitHub Sync

## 1. What It Does
GitHub Sync allows developers to link their repositories to projects. It helps track pull requests, view commit lists on task cards, and close tasks automatically.

## 2. Where to Find It
- Navigate to: **Sidebar → Workspace Settings → Integrations → GitHub**
- Direct route: `/app/github-sync`

## 3. How to Use GitHub Sync
1. Click **Link GitHub Repository**.
2. Install the Enterprise RAG GitHub App on your repo.
3. Link the repository to your Enterprise RAG project boards.

## 4. Custom Templates
In Enterprise RAG, custom notification templates can be used to format messages pushed to developer channels:
- **Format for Upload**: Custom templates must be uploaded in **HTML or Markdown format**.
- **Where to Upload**: Go to **Workspace Settings → Templates → Upload Template**.

## 5. Common Questions

**Q: In Custom Templates, what format must templates be uploaded in, and where can they be uploaded?**  
A: Custom templates must be uploaded in **HTML or Markdown format**. They are uploaded under **Workspace Settings → Templates → Upload Template**.
