# Slack Integration

## 1. What It Does
The Slack Integration connects your Enterprise RAG workspace to your Slack channels, allowing you to receive real-time updates and perform task actions without leaving your chat window.

## 2. Where to Find It
- Navigate to: **Sidebar → Workspace Settings → Integrations → Slack**
- Direct route: `/app/slack-integration`

## 3. How to Use Slack Integration
1. Click **Connect to Slack** in the Integrations panel.
2. Authorize the Enterprise RAG bot in your Slack workspace.
3. Select which channels receive notifications for task creations, completions, or comments.
4. Use commands like `/enterpriserag create [Task Name]` to create tasks directly from Slack.
