# Task Manager

## 1. What It Does
Task Manager helps you create, organize, and track tasks for a project. It allows you to define workloads, assign duties to team members, set due dates, and specify priority levels. It is useful for staying aligned with project schedules and sprint planning.

## 2. Where to Find It
- Navigate to: **Sidebar → Project Boards → Task Board**
- Direct route: `/app/task-manager`

## 3. How to Use Task Manager
1. Go to the **Task Board** from the sidebar menu.
2. Click on **Create Task** (or the **+** button at the top of any status column).
3. Enter a Task Name.
4. Select the Assignee.
5. Set the Due Date.
6. Select the Priority level.
7. Refine your board view using additional filters such as assignee, due date, priority, or tags.
8. Click **Generate** to add it to the board.
9. Task Manager will display the card in the appropriate status column.
10. Review the cards and sort them by due date, priority level, or creation date.
11. If required, export the tasks as a **CSV** file.

Tasks can also be accessed and quick-edited directly from the **Gantt timeline view**.

## 4. Input Fields
- **Task Name**: Enter the title of the task you want to create.
- **Assignee**: Select the team member to assign the task to.
- **Due Date**: Select the deadline for the task.
- **Priority**: Select the urgency level (Low, Medium, High, Critical).
- **Toggle Filter**: Toggle options like "Hide Completed Tasks" to clear up the active view.

## 5. What You Get (Output)
Task Manager provides:
- A board display categorized by status (`Todo`, `In Progress`, `Under Review`, `Completed`).
- Detailed task cards displaying assignees, due dates, and tags.
- Sorting options by due date, priority, or assignee.
- Exportable lists in **CSV** format.

## 7. Common Questions

**Q: Which fields are required to create a task?**  
A: The required fields are **Task Name**, **Assignee**, and **Due Date**. These are essential to define the ownership and timeline of a task.

**Q: Can I select more than one priority level or assignee filter?**  
A: Yes. You can filter by multiple assignees or priorities at the same time.

**Q: What status columns does the board return?**  
A: The board displays four main statuses: `Todo`, `In Progress`, `Under Review`, and `Completed`.

**Q: Can I export the results?**  
A: Yes. Your task lists can be exported as a **CSV** file for further reporting.
